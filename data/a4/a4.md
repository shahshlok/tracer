### Q1. The Book Class [5 marks]

Write a class `Book` with two private instance variables: `title` (String) and `pages` (int).
1. Write a constructor `public Book(String title, int pages)` that initializes these variables.
2. Write a method `isLong()` that returns `true` if pages > 300.

**Task:** Instantiate a Book with title "Java Programming" and 500 pages, then print if it is long.

**Sample run:**

```text
Book created: Java Programming
Is long? true
```

### Q2. Bank Account Transfer [5 marks]

Write a class `BankAccount` with a `balance` (double).
1. Write a method `transferTo(BankAccount other, double amount)`.
2. The method should withdraw `amount` from the current account and add it to `other`.

**Task:** Create two accounts (A with 100, B with 0). Transfer 50 from A to B. Print both balances.

**Sample run:**

```text
Account A: 50.0
Account B: 50.0
```

### Q3. Object Equality Check [5 marks]

Write a program that manages user IDs. Create a class `User` with a field `id` (String).
1. Create two `User` objects, both with the ID "User123".
2. Write code to check if they represent the **same user** based on their ID content.
3. Print "Same User" or "Different User".

**Sample run:**

```text
Comparing User1("User123") and User2("User123")...
Result: Same User
```
